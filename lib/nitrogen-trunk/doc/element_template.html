Template Element
Description
The template element allows you to apply a header, footer, and other common elements defined to your page.
Templates are defined in the file system as /content/web_content/TEMPLATE_NAME.html See the Nitrogen wiki for more information on creating your own templates.
Attributes
file
atom or string
The name of the template file. If the file is named mytemplate.html, then the value of this attribute should be 'mytemplate'
title
string
The title of the page, as set in the HEAD of the html document.
headline
string
A headline for the page, usually displayed at the top of the screen.
section1
element, or list of elements
Sections 1 through 4 are places to put elements. You can specify either a single element or a list of elements.
section2
See section1.
