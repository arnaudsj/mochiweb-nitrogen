Single Row Element
Description
The single row element is a shortcut for a table with one row.
Commonly used by CSS anti-purists to get simple layouts to work in cross-browser scenarious. See http://giveupandusetables.com/ for more information.
Attributes
cells
One or more tablecell elements.
Set the table cells to be rendered within this row.
